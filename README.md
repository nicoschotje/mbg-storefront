# Mr. Beanie's Greenies — Storefront

Mobile-first customer PWA for Mr. Beanie's Greenies (MBG).

- **Live:** https://mbg-storefront-prod.netlify.app
- **Supabase project:** mrbeanies-prod (`ihnnipynpdtcbdfbpemq`)
- **Dashboard repo:** https://github.com/nicoschotje/mbg-dashboard

### Stack
Vanilla JS · ES modules · PWA · Leaflet · Telegram Mini App SDK · Supabase JS 2.45.4

### Deploy
Push to `main` → Netlify auto-deploys in ~30 seconds.
